## 🚀 快速部署（推荐）

### 方式 1：使用一键部署脚本（最简单）

```bash
# 1. 复制项目到 CentOS
scp -r /path/to/xiaomusic root@your_centos_ip:/opt/xiaomusic

# 2. SSH 连接到 CentOS
ssh root@your_centos_ip

# 3. 进入项目目录
cd /opt/xiaomusic

# 4. 运行部署脚本
bash deploy.sh

# 5. 编辑配置文件
vi /xiaomusic_conf/config.json

# 6. 重启容器
docker restart xiaomusic
```

**脚本会自动完成：**
- ✅ 检查 Docker 环境
- ✅ 创建数据目录
- ✅ 复制配置文件
- ✅ 拉取 Docker 镜像
- ✅ 启动容器
- ✅ 验证部署

## 📝 手动部署步骤

### 第 1 步：复制项目文件

```bash
# 在本地机器上
scp -r /path/to/xiaomusic root@your_centos_ip:/opt/xiaomusic

# 验证文件
ssh root@your_centos_ip "ls -la /opt/xiaomusic"
```

### 第 2 步：创建数据目录

```bash
ssh root@your_centos_ip << 'EOF'
sudo mkdir -p /xiaomusic_music
sudo mkdir -p /xiaomusic_conf
sudo chmod -R 755 /xiaomusic_music
sudo chmod -R 755 /xiaomusic_conf
echo "✓ 数据目录创建完成"
EOF
```

### 第 3 步：配置文件

```bash
ssh root@your_centos_ip << 'EOF'
# 复制示例配置
cp /opt/xiaomusic/config-example.json /xiaomusic_conf/config.json

# 编辑配置（使用 sed 快速修改）
sed -i 's/"account": ""/"account": "your_xiaomi_account@example.com"/' /xiaomusic_conf/config.json
sed -i 's/"password": ""/"password": "your_password"/' /xiaomusic_conf/config.json
sed -i 's/"hostname": "http:\/\/192.168.2.5"/"hostname": "http:\/\/your_centos_ip"/' /xiaomusic_conf/config.json

echo "✓ 配置文件已更新"
EOF
```

### 第 4 步：拉取 Docker 镜像

```bash
ssh root@your_centos_ip << 'EOF'
docker pull hanxi/xiaomusic:latest
docker images | grep xiaomusic
echo "✓ 镜像拉取完成"
EOF
```

### 第 5 步：启动容器

```bash
ssh root@your_centos_ip << 'EOF'
cd /opt/xiaomusic
docker-compose up -d
sleep 10
docker ps | grep xiaomusic
echo "✓ 容器已启动"
EOF
```

### 第 6 步：验证部署

```bash
# 获取 CentOS IP
CENTOS_IP=$(ssh root@your_centos_ip "hostname -I | awk '{print \$1}'")

# 测试 API
curl http://$CENTOS_IP:58090/getversion

# 访问 Web 界面
echo "打开浏览器访问: http://$CENTOS_IP:58090"
```

## 🔧 配置文件详解

编辑 `/xiaomusic_conf/config.json`：

```json
{
  "account": "your_xiaomi_account@example.com",  // 小米账号
  "password": "your_password",                    // 小米密码
  "mi_did": "981195288",                          // 设备 ID（可先不填）
  "music_path": "music",                          // 音乐目录
  "hostname": "http://your_centos_ip",            // 服务器地址
  "port": 8090,                                   // 内部端口
  "public_port": 58090,                           // 外部端口
  "verbose": false,                               // 详细日志
  "disable_httpauth": true,                       // 禁用 HTTP 认证
  "enable_config_example": true                   // 启用示例配置
}
```

## 🎵 上传音乐

### 方法 1：使用 SCP（推荐）

```bash
# 在本地机器上
scp -r /path/to/music/* root@your_centos_ip:/xiaomusic_music/

# 或使用 rsync（更快）
rsync -avz /path/to/music/ root@your_centos_ip:/xiaomusic_music/
```

### 方法 2：使用 Web 界面

1. 打开 `http://your_centos_ip:58090`
2. 点击"上传"
3. 选择音乐文件
4. 等待上传完成

### 方法 3：直接复制

```bash
ssh root@your_centos_ip << 'EOF'
# 复制音乐文件
cp -r /path/to/music/* /xiaomusic_music/

# 刷新音乐列表
curl -X POST http://localhost:58090/api/music/refreshlist

echo "✓ 音乐已上传"
EOF
```

## 🐛 故障排查

### 问题 1：容器无法启动

```bash
# 查看错误日志
docker logs xiaomusic

# 检查端口是否被占用
sudo netstat -tlnp | grep 58090

# 检查目录权限
ls -la /xiaomusic_conf/
sudo chown -R 1000:1000 /xiaomusic_conf/
```

### 问题 2：设备列表为空

```bash
# 检查配置
cat /xiaomusic_conf/config.json | jq '.account, .password'

# 查看认证日志
docker logs xiaomusic | grep -i "login\|auth"

# 删除认证文件重新登录
rm /xiaomusic_conf/.mi.token
docker restart xiaomusic

# 等待 30 秒后查看日志
sleep 30
docker logs xiaomusic | tail -20
```

### 问题 3：无法访问 Web 界面

```bash
# 检查防火墙
sudo firewall-cmd --list-all

# 开放端口
sudo firewall-cmd --permanent --add-port=58090/tcp
sudo firewall-cmd --reload

# 或使用 iptables
sudo iptables -A INPUT -p tcp --dport 58090 -j ACCEPT
```

### 问题 4：音乐无法播放

```bash
# 检查音乐文件权限
ls -la /xiaomusic_music/

# 刷新音乐列表
curl -X POST http://localhost:58090/api/music/refreshlist

# 查看日志
docker logs xiaomusic | grep -i "music\|library"
```

## 💾 备份和恢复


```bash
# 恢复配置

# 恢复音乐
```
